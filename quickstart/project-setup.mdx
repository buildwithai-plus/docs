## Boilerplate Setup

This guide will walk you through the process of setting up your BuildWithAI boilerplate from scratch, from cloning the repository to configuring your development environment.

### Prerequisites

Before you begin, make sure you have the following installed:

- [Node.js](https://nodejs.org/en/download) (v16 or later)
- [Git](https://git-scm.com/downloads)
- [Windsurf IDE](https://windsurf.com/download/editor)

### Step 1: Create Your Project

1. Open [Windsurf IDE](https://windsurf.com/download/editor)
2. Click the "Open Folder" button
3. Create a new empty folder for your project
4. Select this folder and click "Open"

### Step 2: Get the Boilerplate code

1. Open the terminal in Windsurf by pressing:
   - Windows: `Ctrl+Shift+\`` (backtick)
   - macOS: `Cmd+Shift+\` (backtick)

2. Run the following commands:

```bash
git clone https://github.com/steffensunbranch/buildwithai-boilerplate.git .

git remote remove origin
```

### Step 3: Install Dependencies

Install all required dependencies:

```bash
npm install
```

### Step 4: Create Environment Variables File

The boilerplate includes an example environment file with placeholders for various services:

```bash
cp .env.example .env
```

<Info>
  You'll configure specific environment variables as you set up various features in the following guides.
</Info>
