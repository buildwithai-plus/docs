## Download Windsurf

Check your system requirements and download the installer for your platform:

<Tabs>
  <Tab title="Windows">
    **Requirements**: Windows 10 or later
    
    [Download for Windows](https://windsurf.com/windsurf/download)
  </Tab>
  <Tab title="macOS">
    **Requirements**: OS X Yosemite or later
    
    [Download for macOS](https://windsurf.com/windsurf/download_mac)
  </Tab>
  <Tab title="Linux">
    **Requirements**: Ubuntu 20.04+ (glibc ≥ 2.31, glibcxx ≥ 3.4.26)
    
    Other distributions: glibc ≥ 2.28, glibcxx ≥ 3.4.25
    
    [Download for Linux](https://windsurf.com/windsurf/download_linux)
  </Tab>
</Tabs>

## Install Windsurf

1. Download the installer for your platform from the links above
2. Run the installer and follow your system's standard installation process
3. Launch Windsurf to begin setup

## First-Time Setup

<Steps>
  <Step title="Launch Windsurf">
    Open Windsurf for the first time to begin the onboarding process.
  </Step>
  
  <Step title="Choose Setup Flow">
    Select how you want to configure Windsurf:
    - **Start fresh** - Clean installation with default settings (recommended for beginners)
    - **Import from VS Code** - Migrate your existing VS Code settings, extensions, and configurations
    - **Import from Cursor** - Migrate your existing Cursor settings and configurations
    
    <Info>If you're coming from VS Code or Cursor, importing will save time by transferring your familiar setup</Info>
  </Step>
  
  <Step title="Configure Editor">
    Choose your preferences:
    - **Keybindings**: VS Code (recommended) or Vim
    - **Theme**: Select your preferred color scheme
  </Step>
  
  <Step title="Create Account">
    Sign up for a free Windsurf account or log in with an existing account.
    
    <Warning>A Windsurf account is required to use the IDE</Warning>
  </Step>
</Steps>

<Check>
  You've successfully installed and configured Windsurf IDE! You can now open projects and start developing.
</Check>
