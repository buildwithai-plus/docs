## Prerequisites

Check that your system meets the minimum requirements:

<Tabs>
  <Tab title="Windows">
    Windows 10 or later
  </Tab>
  <Tab title="macOS">
    OS X Yosemite or later
  </Tab>
  <Tab title="Linux">
    Ubuntu 20.04+ (glibc ≥ 2.31, glibcxx ≥ 3.4.26)
    
    Other distributions: glibc ≥ 2.28, glibcxx ≥ 3.4.25
  </Tab>
</Tabs>

## Download Windsurf

Download the installer for your platform:

<Tabs>
  <Tab title="Windows">

  </Tab>
  <Tab title="macOS">
    [Download for macOS](https://windsurf.com/windsurf/download_mac)
  </Tab>
  <Tab title="Linux">
    [Download for Linux](https://windsurf.com/windsurf/download_linux)
  </Tab>

## Install Windsurf

1. Download the installer for your platform from the links above
2. Run the installer and follow your system's standard installation process
3. Launch Windsurf to begin setup

## First-Time Setup

<Steps>
  <Step title="Launch Windsurf">
    Open Windsurf for the first time to begin the onboarding process.
  </Step>
  
  <Step title="Choose Setup Flow">
    Select how you want to configure Windsurf:
    - **Start fresh** - Clean installation with default settings (recommended for beginners)
    - **Import from VS Code** - Migrate your existing VS Code settings, extensions, and configurations
    - **Import from Cursor** - Migrate your existing Cursor settings and configurations
    
    <Info>If you're coming from VS Code or Cursor, importing will save time by transferring your familiar setup</Info>
  </Step>
  
  <Step title="Configure Editor">
    Choose your preferences:
    - **Keybindings**: VS Code (recommended) or Vim
    - **Theme**: Select your preferred color scheme
  </Step>
  
  <Step title="Create Account">
    Sign up for a free Windsurf account or log in with an existing account.
    
    <Warning>A Windsurf account is required to use the IDE</Warning>
  </Step>
  
  <Step title="Optional: Install Windsurf in PATH">
    During setup, you'll have the option to install Windsurf in your system PATH.
    
    **Benefits of installing in PATH:**
    - Open projects from command line with `windsurf .`
    - Launch Windsurf from any terminal or command prompt
    - Integrate with other development tools
    
    <Tip>Recommended for developers who frequently use the command line</Tip>
  </Step>
</Steps>

## Explore Windsurf Features

After installation, you can:

- **Access Cascade**: Use the AI assistant panel for code generation and help
- **Open projects**: Use the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`) to open folders or remote servers
- **Generate projects**: Use AI to create new projects
- **Configure settings**: Customize Windsurf to your preferences

<Info>Windsurf includes built-in AI assistance and doesn't support extensions from external marketplaces</Info>

## Updates

Windsurf will notify you when updates are available:

- Click **"Restart to Update"** when prompted
- Manually check: Profile dropdown > **"Check for Updates"**
- Or use Command Palette > **"Check for Updates"**

<Accordion title="Need to restart onboarding">
  Open Command Palette and search for "Reset Onboarding" to restart the setup process.
</Accordion>
