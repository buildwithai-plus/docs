## Download Cursor

Visit [cursor.com/downloads](https://cursor.com/downloads) and download the version for your system (Windows, macOS, or Linux).

## Install Cursor

1. Run the downloaded installer
2. Launch Cursor when installation is complete

## First-Time Setup

<Steps>
  <Step title="Launch Cursor">
    Open Cursor for the first time. You'll see a welcome screen with setup options.
  </Step>
  
  <Step title="Choose Keyboard Shortcuts">
    Pick how you want to control Cursor:
    - **VS Code** - If you've used VS Code before (recommended)
    - **Default** - If you're new to code editors
    
    <Tip>Most developers choose VS Code shortcuts since they're widely used</Tip>
  </Step>
  
  <Step title="Pick a Theme">
    Choose a color scheme you like. You can always change this later.
  </Step>
  
  <Step title="Set Terminal Preferences">
    Choose your preferred terminal settings. The default options work fine for most people.
  </Step>
  
  <Step title="Create Account">
    Sign up for a free Cursor account to unlock AI features.
  </Step>
</Steps>

<Check>
  You've successfully installed and configured Cursor IDE! You can now open projects and start developing.
</Check>
