## Explore Cursor Features

After installation, you can:

- **AI Code Assistance**: Get intelligent code suggestions and completions
- **Codebase Indexing**: Cursor automatically indexes your projects for better AI understanding
- **Multi-language Support**: Works with all major programming languages including TypeScript/JavaScript
- **VS Code Compatibility**: Import extensions and settings from VS Code

<Note>When you open a project, Cursor will automatically start indexing your codebase. This process takes 1-15 minutes depending on project size</Note>

## Essential Keyboard Shortcuts

Learn these shortcuts to boost your productivity with Cursor:

### AI Features
- `Ctrl+L` (Windows/Linux) or `Cmd+L` (macOS) - Add selection to new chat
- `Ctrl+Shift+L` - Add selected code to chat
- `Ctrl+K` - Open inline edit
- `Ctrl+/` - Loop between AI models
- `Tab` - Accept code suggestion
- `Ctrl+Enter` - Search codebase in chat

### Navigation & Interface
- `Ctrl+P` / `Cmd+P` - Open file picker (quickly search for a file to open)
- `Ctrl+Shift+P` / `Cmd+Shift+P` - Open command palette
- `Ctrl+.` / `Cmd+.` - Open mode menu
- `Ctrl+I` / `Cmd+I` - Toggle sidepanel
- `Ctrl+T` / `Cmd+T` - Open new chat tab
- `Ctrl+[` and `Ctrl+]` - Navigate between chat tabs

<Tip>Use `Ctrl+K` for quick inline edits and `Ctrl+L` to start conversations about your code</Tip>

## @-Symbols for Context

Use @-symbols in chat to provide precise context to Cursor's AI:

- **@Files** - Reference specific project files
- **@Folders** - Reference entire directories (like `/components` or `/lib`)
- **@Code** - Reference specific functions or components
- **@Git** - Access git history and recent changes
- **@Recent Changes** - Track your latest modifications

### Usage Tips
- Type `@` in chat to see available symbols
- Use arrow keys to navigate suggestions
- Press `Enter` to select
- Combine multiple @-symbols for better context

<Info>Example: "@Files app/(tabs)/index.tsx @Recent Changes" gives Cursor context about your main screen and recent edits</Info>
