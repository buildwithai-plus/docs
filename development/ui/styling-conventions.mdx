# Styling Conventions

This guide outlines the styling conventions used in BuildWithAI+ applications.

## NativeWind Integration

BuildWithAI+ uses NativeWind (Tailwind CSS for React Native) as the primary styling solution.

## Organization

### Component-Specific Styles

```tsx
// Example of component-specific styling
export function Button({ children, variant = 'primary' }) {
  return (
    <Pressable
      className={cn(
        "py-3 px-4 rounded-lg",
        variant === 'primary' && "bg-primary text-white",
        variant === 'secondary' && "bg-gray-200 text-gray-800"
      )}
    >
      <Text className="text-center font-medium">{children}</Text>
    </Pressable>
  );
}
```
