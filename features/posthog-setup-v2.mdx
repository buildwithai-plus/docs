<Note>
  This guide sets up PostHog via the official AI wizard. The wizard automatically configures your app with all necessary settings.
</Note>

## Step 1: Run the PostHog Wizard

<Steps>
  <Step title="Launch the wizard">
    In your project root directory, run the following command:

    ```bash
    npx @posthog/wizard --integration react-native
    ```
  </Step>

  <Step title="Set up your account">
    During the wizard setup:
    - Choose <b>Sign up</b> if you don't have a PostHog account yet
    - If you already have an account, choose <b>Log in</b>
  </Step>

  <Step title="Select your data region">
    Choose where your analytics data will be stored:
    - <b>EU</b>: Frankfurt, Germany (GDPR-compliant, faster for European users)
    - <b>US</b>: Virginia, USA (faster for North American users)
  </Step>
</Steps>

## Step 2: Test the Implementation

<Steps>
  <Step title="Implementation">
    Follow the implementation instructions in the `integrations/posthog/README.md` file included in your boilerplate codebase. This file contains all necessary steps for testing PostHog in your application, including the test button component.
  </Step>

  <Step title="Test the integration">
    Once you've followed the README instructions:
    1. Restart your app
    2. Use the test button component as described in the README
    3. Log in to your PostHog dashboard
    4. Go to <b>Live Events</b> section
    5. Confirm that your test events appear in the dashboard
  </Step>
</Steps>
