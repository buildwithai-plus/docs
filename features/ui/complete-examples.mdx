## Example 1: Login Form

This example shows a complete login form with email and password inputs, a submit button, and links for forgotten password and sign up.

```tsx
// ðŸ“± Complete login form with validation state
import React, { useState } from 'react';
import { View, TouchableOpacity } from 'react-native';
import { Text } from '~/components/ui/text';
import { Input } from '~/components/ui/input';
import { Button } from '~/components/ui/button';

export function LoginScreen() {
  const [email, setEmail] = useState('');
  const [password, setPassword] = useState('');
  const [isLoading, setIsLoading] = useState(false);
  const [error, setError] = useState('');
  
  const handleLogin = () => {
    setError('');
    setIsLoading(true);
    
    // Simulate API call
    setTimeout(() => {
      if (email === 'test@example.com' && password === 'password') {
        // Success - would navigate in a real app
        console.log('Login successful');
      } else {
        setError('Invalid email or password');
      }
      setIsLoading(false);
    }, 1000);
  };
  
  return (
    <View className="flex-1 bg-background p-6">
      {/* Theme system provides bg-background color */}
      
      <View className="mb-8">
        {/* NativeWind provides spacing with mb-8 */}
        <Text className="text-3xl font-bold text-foreground">Welcome back</Text>
        <Text className="text-muted-foreground mt-2">
          Enter your credentials to access your account
        </Text>
      </View>
      
      {error ? (
        <View className="bg-destructive/10 p-3 rounded-md mb-4">
          {/* NativeWind provides styling and theme colors */}
          <Text className="text-destructive">{error}</Text>
        </View>
      ) : null}
      
      <View className="gap-4">
        <View>
          <Text className="text-sm font-medium mb-1.5">Email</Text>
          <Input
            placeholder="Enter your email"
            keyboardType="email-address"
            value={email}
            onChangeText={setEmail}
            autoCapitalize="none"
          />
          {/* React Native Reusables provides the Input component */}
        </View>
        
        <View>
          <Text className="text-sm font-medium mb-1.5">Password</Text>
          <Input
            placeholder="Enter your password"
            secureTextEntry
            value={password}
            onChangeText={setPassword}
          />
        </View>
        
        <TouchableOpacity>
          <Text className="text-primary text-right text-sm">
            Forgot password?
          </Text>
        </TouchableOpacity>
        
        <Button 
          onPress={handleLogin} 
          disabled={isLoading}
          className="mt-2"
        >
          <Text className="text-primary-foreground font-medium">
            {isLoading ? 'Signing in...' : 'Sign in'}
          </Text>
        </Button>
        
        <View className="flex-row justify-center mt-6">
          <Text className="text-muted-foreground">
            Don't have an account?{' '}
          </Text>
          <TouchableOpacity>
            <Text className="text-primary font-medium">Sign up</Text>
          </TouchableOpacity>
        </View>
      </View>
    </View>
  );
}
```

### What's Happening in This Example

- **Theme System** provides semantic colors like `bg-background`, `text-primary`, and `text-destructive`
- **NativeWind** handles layout with `flex-1`, `p-6`, `mb-8`, and other utility classes
- **React Native Reusables** provides the `Input` and `Button` components
- State management handles form validation and loading states

## Example 2: Profile Card

This example shows a user profile card with avatar, stats, and action buttons.

```tsx
// ðŸ“± User profile card with stats and actions
import React from 'react';
import { View, Image, TouchableOpacity } from 'react-native';
import { Text } from '~/components/ui/text';
import { Card, CardHeader, CardContent, CardFooter } from '~/components/ui/card';
import { Button } from '~/components/ui/button';

export function ProfileCard({ user }) {
  // In a real app, you'd pass a user object as a prop
  const userData = user || {
    name: 'Sarah Johnson',
    role: 'Product Designer',
    avatar: 'https://i.pravatar.cc/300',
    followers: 2453,
    following: 348,
    projects: 29
  };
  
  return (
    <View className="p-4">
      <Card>
        {/* React Native Reusables provides the Card component */}
        
        <CardHeader className="items-center pb-2">
          <View className="mb-3">
            <Image
              source={{ uri: userData.avatar }}
              className="w-24 h-24 rounded-full"
              // NativeWind styles the image dimensions and border radius
            />
          </View>
          
          <Text className="text-xl font-bold text-center">
            {userData.name}
          </Text>
          <Text className="text-muted-foreground text-center">
            {userData.role}
          </Text>
        </CardHeader>
        
        <CardContent>
          <View className="flex-row justify-between py-2 border-y border-border">
            {/* Theme system provides border-border color */}
            
            <View className="items-center flex-1">
              <Text className="font-bold">{userData.followers.toLocaleString()}</Text>
              <Text className="text-xs text-muted-foreground">Followers</Text>
            </View>
            
            <View className="items-center flex-1">
              <Text className="font-bold">{userData.following.toLocaleString()}</Text>
              <Text className="text-xs text-muted-foreground">Following</Text>
            </View>
            
            <View className="items-center flex-1">
              <Text className="font-bold">{userData.projects}</Text>
              <Text className="text-xs text-muted-foreground">Projects</Text>
            </View>
          </View>
        </CardContent>
        
        <CardFooter className="flex-row gap-2">
          <Button 
            variant="outline" 
            className="flex-1"
            onPress={() => console.log('Message')}
          >
            <Text>Message</Text>
          </Button>
          
          <Button 
            className="flex-1"
            onPress={() => console.log('Follow')}
          >
            <Text className="text-primary-foreground">Follow</Text>
          </Button>
        </CardFooter>
      </Card>
    </View>
  );
}
```

### What's Happening in This Example

- **Theme System** provides semantic colors like `text-muted-foreground` and `border-border`
- **NativeWind** handles layout with `flex-row`, `items-center`, and sizing with `w-24 h-24`
- **React Native Reusables** provides the `Card`, `CardHeader`, `CardContent`, `CardFooter`, and `Button` components
- Component composition creates a clean, maintainable structure

## Example 3: Settings List

This example shows a settings screen with different types of options.

```tsx
// ðŸ“± Settings screen with various option types
import React, { useState } from 'react';
import { View, ScrollView, TouchableOpacity } from 'react-native';
import { Text } from '~/components/ui/text';
import { Switch } from '~/components/ui/switch';
import { ChevronRight } from '~/components/icons'; // Assume you have icons

export function SettingsScreen() {
  const [notifications, setNotifications] = useState(true);
  const [darkMode, setDarkMode] = useState(false);
  const [locationServices, setLocationServices] = useState(true);
  
  return (
    <ScrollView className="flex-1 bg-background">
      {/* Theme system provides bg-background color */}
      
      <View className="p-4">
        <Text className="text-2xl font-bold mb-6">Settings</Text>
        
        {/* Section: App Preferences */}
        <View className="mb-8">
          <Text className="text-sm font-medium text-muted-foreground mb-2 px-2">
            APP PREFERENCES
          </Text>
          
          <View className="bg-card rounded-lg overflow-hidden">
            {/* Theme system provides bg-card color */}
            
            {/* Toggle option */}
            <View className="flex-row justify-between items-center p-4 border-b border-border">
              <View>
                <Text className="font-medium">Notifications</Text>
                <Text className="text-sm text-muted-foreground">
                  Receive app notifications
                </Text>
              </View>
              <Switch 
                value={notifications} 
                onValueChange={setNotifications} 
              />
              {/* React Native Reusables provides the Switch component */}
            </View>
            
            {/* Toggle option */}
            <View className="flex-row justify-between items-center p-4 border-b border-border">
              <View>
                <Text className="font-medium">Dark Mode</Text>
                <Text className="text-sm text-muted-foreground">
                  Use dark theme
                </Text>
              </View>
              <Switch 
                value={darkMode} 
                onValueChange={setDarkMode} 
              />
            </View>
            
            {/* Toggle option */}
            <View className="flex-row justify-between items-center p-4">
              <View>
                <Text className="font-medium">Location Services</Text>
                <Text className="text-sm text-muted-foreground">
                  Use your current location
                </Text>
              </View>
              <Switch 
                value={locationServices} 
                onValueChange={setLocationServices} 
              />
            </View>
          </View>
        </View>
        
        {/* Section: Account */}
        <View className="mb-8">
          <Text className="text-sm font-medium text-muted-foreground mb-2 px-2">
            ACCOUNT
          </Text>
          
          <View className="bg-card rounded-lg overflow-hidden">
            {/* Link option */}
            <TouchableOpacity 
              className="flex-row justify-between items-center p-4 border-b border-border"
              onPress={() => console.log('Navigate to profile')}
            >
              <Text className="font-medium">Edit Profile</Text>
              <ChevronRight className="text-muted-foreground" size={20} />
            </TouchableOpacity>
            
            {/* Link option */}
            <TouchableOpacity 
              className="flex-row justify-between items-center p-4 border-b border-border"
              onPress={() => console.log('Navigate to password')}
            >
              <Text className="font-medium">Change Password</Text>
              <ChevronRight className="text-muted-foreground" size={20} />
            </TouchableOpacity>
            
            {/* Link option */}
            <TouchableOpacity 
              className="flex-row justify-between items-center p-4"
              onPress={() => console.log('Navigate to privacy')}
            >
              <Text className="font-medium">Privacy Settings</Text>
              <ChevronRight className="text-muted-foreground" size={20} />
            </TouchableOpacity>
          </View>
        </View>
        
        {/* Danger Zone */}
        <View>
          <Text className="text-sm font-medium text-muted-foreground mb-2 px-2">
            DANGER ZONE
          </Text>
          
          <View className="bg-card rounded-lg overflow-hidden">
            <TouchableOpacity 
              className="p-4"
              onPress={() => console.log('Log out')}
            >
              <Text className="font-medium text-primary">Log Out</Text>
            </TouchableOpacity>
            
            <TouchableOpacity 
              className="p-4 border-t border-border"
              onPress={() => console.log('Delete account')}
            >
              <Text className="font-medium text-destructive">Delete Account</Text>
              {/* Theme system provides text-destructive color */}
            </TouchableOpacity>
          </View>
        </View>
      </View>
    </ScrollView>
  );
}
```

### What's Happening in This Example

- **Theme System** provides semantic colors like `bg-background`, `bg-card`, `text-destructive`, and `border-border`
- **NativeWind** handles layout with `flex-row`, `justify-between`, and spacing with `p-4`, `mb-8`
- **React Native Reusables** provides the `Switch` component
- Consistent patterns create a professional, maintainable settings screen

## Putting It All Together

These examples demonstrate how the three styling systems work together to create professional, consistent interfaces:

1. **Theme System** provides the color palette that adapts to light/dark mode
2. **NativeWind** provides the utility classes for layout, spacing, and applying theme colors
3. **React Native Reusables** provides pre-built components that follow best practices

By combining these systems, you can create any UI pattern while maintaining consistency and following best practices.
