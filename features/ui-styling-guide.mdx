# UI & Styling Guide

Your app comes with everything you need to build beautiful interfaces. Don't worry about complex styling - the AI will help you with that! Here's a quick overview of what's available.

## Quick Start with NativeWind

Style your components using simple class names instead of complex style objects:

```tsx
// Add padding and make text bigger
<View className="p-4">
  <Text className="text-lg font-bold">Hello World</Text>
</View>

// Put things side by side
<View className="flex-row items-center gap-3">
  <Text>ðŸ‘‹</Text>
  <Text>Welcome!</Text>
</View>
```

**Most useful classes:**
- `p-4` - Add padding (space inside)
- `flex-row` - Put items side by side
- `text-lg` - Make text larger
- `bg-primary` - Use your app's main color
- `rounded-lg` - Add rounded corners

**Pro tip:** Use theme colors like `bg-primary` and `text-foreground` instead of hardcoded colors like `bg-white`. They automatically work in both light and dark mode!

### Quick Component Examples

**Text:**
```tsx
import { Text } from "~/components/ui/text"

<Text>Normal text</Text>
<Text variant="muted">Lighter text</Text>
```

**Button:**
```tsx
import { Button } from "~/components/ui/button"

<Button>
  <Text>Click me</Text>
</Button>
```

**Card:**
```tsx
import { Card, CardContent } from "~/components/ui/card"

<Card>
  <CardContent>
    <Text>Card content</Text>
  </CardContent>
</Card>
```

**Need to customize?** Add `className` to any component to override the default styling. The AI can help you with the right class names!

## Adding Icons

Your app comes with many icons ready to use. Need more? Here's how:

**Example: Adding a Heart icon**

1. **Create** `~/lib/icons/Heart.tsx`:
```tsx
import { Heart } from 'lucide-react-native';
import { iconWithClassName } from './iconWithClassName';
iconWithClassName(Heart);
export { Heart };
```

2. **Use it anywhere:**
```tsx
import { Heart } from "~/lib/icons/Heart"

<Heart className="text-primary" size={24} />
```

**Browse all icons:** [lucide.dev/icons](https://lucide.dev/icons/)

## Quick Tips

1. **Always import from `~/components/ui/`** - Don't use React Native's built-in components
2. **Use theme colors** - `bg-primary` instead of `bg-blue-500`
3. **Test both themes** - Check your UI in light and dark mode
4. **Install components first** - Check what's available before building custom UI
