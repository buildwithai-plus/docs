## What's Included in the Boilerplate

BuildWithAI+ comes packed with pre-configured features, integrations, and tools that would typically take weeks to set up from scratch. Here's a breakdown of what's included out of the box:

### Core Foundation

<AccordionGroup>
  <Accordion icon="react" title="React Native & Expo">
    Built on the latest stable versions of React Native and Expo, providing a solid foundation for cross-platform mobile development. Includes Expo's development tools, build services, and update mechanisms.
  </Accordion>
  
  <Accordion icon="code" title="TypeScript">
    Fully typed codebase with TypeScript for improved developer experience, code quality, and maintainability.
  </Accordion>
  
  <Accordion icon="sitemap" title="Expo Router">
    File-based routing system with automatic deep linking support, making navigation simple and intuitive.
  </Accordion>
  
  <Accordion icon="palette" title="NativeWind (Tailwind CSS)">
    Complete styling system using NativeWind, allowing you to use Tailwind CSS in React Native with responsive design support and dark mode.
  </Accordion>
</AccordionGroup>

### Authentication & Backend

<AccordionGroup>
  <Accordion icon="lock" title="Supabase Authentication">
    Ready-to-use authentication with multiple methods:
    - Email/Password login
    - Phone number OTP verification
    - Social logins (Google, Apple)
    - Complete onboarding flows
  </Accordion>
  
  <Accordion icon="database" title="Supabase Backend Integration">
    Database and backend services integration with Supabase, including:
    - User management
    - Row-level security policies
    - API integration with your app
  </Accordion>
</AccordionGroup>

### Monetization & Engagement

<AccordionGroup>
  <Accordion icon="credit-card" title="In-App Purchases">
    Complete in-app purchase infrastructure with:
    - RevenueCat integration for purchase management
    - Superwall for creating paywalls and upsell opportunities
    - Subscription management and entitlement checking
  </Accordion>
  
  <Accordion icon="bell" title="Push Notifications">
    End-to-end push notification system with:
    - Firebase Cloud Messaging for Android
    - Apple Push Notification Service for iOS
    - Permission handling
    - Token management
    - Local and remote notification support
  </Accordion>
  
  <Accordion icon="rectangle-ad" title="AdMob">
    Mobile advertising setup with:
    - Banner ads
    - Interstitial ads
    - Rewarded ads
    - Ad consent management
  </Accordion>
</AccordionGroup>
